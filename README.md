# Fernandino Assessment Skill Training - Center

Manages trainee registration, certificate generation, and facial-recognition attendance with AI-powered job matching to connect certified trainees to relevant job opportunities.

## Tech Stack

MongoDB     -   Database
ExpressJS   -   Backend Framework
React       -   Frontend Library
NodeJS      -   Server Runtime

## Features

1. Trainee registration and profiling
2. Facial recognition attendance
3. Ceritifcate PDF generation
4. Role-based Access (Admin, Company, Trainee)
5. AI-based job matching
